
Aida's Boutique is a full-stack MERN e-commerce application built for a women's clothing store. It allows customers to browse products, manage their cart and wishlist, place orders, and make payments online. It also includes an admin dashboard for managing products, orders, and users.

This project was built as a portfolio project to practice building a complete e-commerce application from frontend to backend.

## Features

### Customer

- Create an account and log in
- Browse products
- Search and filter products
- View product details
- Add products to cart
- Manage wishlist
- Place orders
- Pay using Razorpay
- View previous orders
- Fully responsive design

### Admin

- Dashboard
- Add new products
- Edit existing products
- Activate or deactivate products
- Upload product images with Cloudinary
- Manage customer orders
- Update order status
- View registered users
- Change user roles

## Installation

Clone the repository

```bash
git clone https://github.com/sajil-s/Aidas-boutique.git
```

Install dependencies

```bash
cd frontend
npm install

cd ../backend
npm install
```

---


## Run Locally

Backend

```bash
npm run dev
```

Frontend

```bash
npm run dev
```

---

## Deployment

- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
- Image Storage: Cloudinary
